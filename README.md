# README

## Description
This project is a **.NET 8 Aspire** application with a web frontend.  
This application is an assignment for **Quad Solutions**.

## Starting the application
1. Open **Visual Studio**
2. Open the `.sln` file of the project
3. Start the solution

## Open the web application
- After starting, multiple services will start using Aspire
- Find the **web frontend** in the Aspire dashboard or in the output
- Open the **URL** of the web frontend in your browser
- The web application will open and u need to open the trivia game tab in the navbar

## Testing
Use the web application to test the available functionality by answering the questions.
